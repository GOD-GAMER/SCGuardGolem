# SecurityCraft Guard Golem Addon

A **SecurityCraft** addon that adds a Security Guard Golem entity with patrol routes,
badge-based player detection, configurable threat modes, upgrades, and player target lists.

---

## Download & Install

Each Minecraft version has its own branch with a complete standalone addon.
Download the JAR for your version from [Releases](https://github.com/GOD-GAMER/SCGuardGolem/releases)
and place it in your `.minecraft/mods/` folder alongside SecurityCraft.

## Features

- **Keycard Conversion** - Right-click an Iron Golem with a SecurityCraft keycard to create a Security Guard Golem
- **Patrol System** - Define waypoints and set patrol speed
- **Threat Modes** - WARN (chat message), FOLLOW (shadow), or ATTACK (combat)
- **Upgrades** - Damage, Speed, and Detection range (5 levels each)
- **Player Lists** - Ignore list and always-attack list
- **In-Game Manual** - Book item with full usage guide
- **Commands** - Complete `/scgolem` command tree for all features

---

## Quick Start

1. Build a vanilla Iron Golem
2. Hold any SecurityCraft Keycard
3. Right-click the Iron Golem
4. Use `/scgolem` commands to configure patrols, threat modes, and upgrades

See the branch README for your version for full documentation.
